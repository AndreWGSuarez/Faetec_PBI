Dicas práticas de configuração no Power BI
1
Organize o modelo antes de criar visuais Modelo
Crie uma tabela separada chamada _Métricas para todas as suas medidas DAX — nunca as deixe soltas na tabela principal. Oculte colunas auxiliares com clique direito → "Ocultar na exibição de relatório".
2
Use um tema JSON personalizado Design
Exporte um tema JSON em Exibição → Temas → Personalizar tema atual e defina as 6 cores da paleta. Assim todos os visuais novos herdam automaticamente as cores certas sem ajuste manual.
3
Slicers sincronizados entre páginas Interatividade
Vá em Exibição → Sincronizar segmentações e marque os filtros de Gender, School_Type e Faixa de Performance para sincronizar nas 3 páginas. O usuário filtra uma vez e a seleção aplica ao relatório inteiro.
4
Formatação condicional nos KPIs Design
Nos cartões KPI, use Formatação condicional → Cor da fonte com regras: score acima da média → verde teal (#1D9E75), abaixo → coral (#D85A30).
5
Drillthroughs para análise detalhada Interatividade
Crie uma página de drillthrough (clique direito em página → "Adicionar página de drillthrough"). Quando o usuário clica em um ponto no gráfico de dispersão, é levado automaticamente aos detalhes daquele grupo de alunos.
6
Visuais do AppSource recomendados Plugins
Instale pelo AppSource (gratuitos): Box and Whisker Chart para o box plot de Motivation_Level, Radar Chart da Microsoft para o perfil do aluno, e Correlation Plot para o mapa de calor.
7
Power Query — limpeza recomendada ETL
No Power Query, crie uma coluna Faixa_Score com Adicionar coluna → Coluna condicional. Converta Parental_Involvement, Motivation_Level e Family_Income em colunas ordinais (Low=1, Medium=2, High=3) para ordenação correta nos eixos.
