Medidas DAX recomendadas

Organize em uma tabela de medidas chamada _Métricas para manter o modelo limpo

1. Métricas básicas de desempenho Base
Média Exam Score = AVERAGE(StudentPerformance[Exam_Score]) Mediana Exam Score = MEDIAN(StudentPerformance[Exam_Score]) Score Máximo = MAX(StudentPerformance[Exam_Score]) Score Mínimo = MIN(StudentPerformance[Exam_Score]) Desvio Padrão Score = STDEV.P(StudentPerformance[Exam_Score]) Total Alunos = COUNTROWS(StudentPerformance)
2. Classificação de performance Coluna calculada
Faixa de Performance = SWITCH(TRUE(), StudentPerformance[Exam_Score] >= 80, "Alto (≥80)", StudentPerformance[Exam_Score] >= 70, "Médio-Alto (70-79)", StudentPerformance[Exam_Score] >= 60, "Médio (60-69)", "Baixo (<60)" )
3. % de alunos por faixa Base
% Alto Desempenho = DIVIDE( COUNTROWS(FILTER(StudentPerformance, StudentPerformance[Exam_Score] >= 80)), [Total Alunos], 0 ) % Baixo Desempenho = DIVIDE( COUNTROWS(FILTER(StudentPerformance, StudentPerformance[Exam_Score] < 60)), [Total Alunos], 0 )
4. Comparação por grupo (Parental Involvement) Análise cruzada
Score Médio - Alto Envolvimento Parental = CALCULATE( [Média Exam Score], StudentPerformance[Parental_Involvement] = "High" ) Gap Envolvimento Parental = [Score Médio - Alto Envolvimento Parental] - CALCULATE( [Média Exam Score], StudentPerformance[Parental_Involvement] = "Low" )
5. Índice de eficiência de estudo Coluna calculada
Eficiência Estudo = DIVIDE( StudentPerformance[Exam_Score], StudentPerformance[Hours_Studied], 0 )
6. Score médio com acesso à internet vs sem Análise cruzada
Score Médio c/ Internet = CALCULATE([Média Exam Score], StudentPerformance[Internet_Access] = "Yes") Score Médio s/ Internet = CALCULATE([Média Exam Score], StudentPerformance[Internet_Access] = "No") Gap Internet = [Score Médio c/ Internet] - [Score Médio s/ Internet]
7. Análise de frequência e score Base
Média Score - Alta Frequência = CALCULATE([Média Exam Score], StudentPerformance[Attendance] >= 90) Média Score - Baixa Frequência = CALCULATE([Média Exam Score], StudentPerformance[Attendance] < 75) % Alunos Alta Frequência = DIVIDE( COUNTROWS(FILTER(StudentPerformance, StudentPerformance[Attendance] >= 90)), [Total Alunos], 0 )
8. Ranking de fatores Avançado
Score vs Média Geral = [Média Exam Score] - CALCULATE([Média Exam Score], ALL(StudentPerformance)) Score Alunos Escola Privada = CALCULATE([Média Exam Score], StudentPerformance[School_Type] = "Private") Score Alunos Escola Pública = CALCULATE([Média Exam Score], StudentPerformance[School_Type] = "Public")

Perguntas-chave para a análise

Questões a serem respondidas com o dashboard e as métricas DAX criadas

Desempenho geral
1
Qual é a distribuição das notas? Os alunos tendem a ter desempenho alto, médio ou baixo?
→ Histograma do Exam_Score + Rosca de faixa de performance
2
Há diferença significativa de desempenho entre alunos de escolas públicas e privadas?
→ Medida "Score Alunos Escola Privada vs Pública" + Barras agrupadas
3
Existem diferenças de desempenho entre gêneros? O padrão se mantém por tipo de escola?
→ Barras segmentadas por Gender + Slicer School_Type
Fatores de aprendizagem
4
Quantas horas de estudo separam os alunos de alto desempenho dos de baixo? Existe um limiar crítico?
→ Dispersão Hours_Studied vs Exam_Score + linha de tendência
5
Alunos com alta frequência (≥90%) têm notas significativamente maiores?
→ Medida "Score Alunos Alta Frequência" comparada à média geral
6
Sessões de tutoria compensam uma motivação baixa?
→ Tabela cruzada Motivation_Level × Tutoring_Sessions com Média Exam Score
7
Qual a influência das notas anteriores (Previous_Scores) no Exam_Score atual?
→ Gráfico de dispersão Previous_Scores vs Exam_Score
Fatores socioeconômicos e contextuais
8
O nível de renda familiar afeta diretamente o desempenho? O acesso a recursos media esse efeito?
→ Barras Family_Income vs Score + slicer Access_to_Resources
9
Alunos com acesso à internet têm desempenho melhor?
→ Medida "Gap Internet"
10
Quais combinações de fatores produzem os maiores Exam_Scores médios?
→ Tabela com top 10 combinações de perfil × Score médio
11
Alunos com deficiência de aprendizagem têm desempenho muito diferente? Quais fatores ajudam a mitigar esse impacto?
→ Filtro Learning_Disabilities + análise de fatores protetores
12
A influência dos pares (Peer_Influence) tem impacto mensurável no Exam_Score?
→ Barras Peer_Influence vs Score + cruzamento com Motivation_Level
